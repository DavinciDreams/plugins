# Vercel Generative UI Command

Main command for the Vercel Generative UI team.

## Usage

```
/vercel-gen-ui [subcommand]
```

## Subcommands

- `build` - Build a new generative UI application
- `add-ai` - Add AI features to existing app
- `deploy` - Deploy to Vercel
- `component` - Create AI-powered component
- `stream` - Add streaming to a feature

## Examples

```
/vercel-gen-ui build my-gen-ui-app
/vercel-gen-ui add-ai --feature chat
/vercel-gen-ui component streaming-chat
```
