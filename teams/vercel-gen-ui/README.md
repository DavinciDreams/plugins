# Vercel Generative UI Team

A specialized team for building generative UI applications using Vercel's ecosystem.

## Team Members

- **Vercel Deployment Specialist**: Handles deployment, environment configuration, and CI/CD pipelines
- **Next.js Developer**: Builds the core application with Next.js, focusing on generative UI patterns
- **Vercel AI SDK Expert**: Implements AI features, streaming, tool calling, and structured output
- **Vercel AI Elements Developer**: Creates and integrates AI-powered UI components

## Specialization

This team specializes in:
- Generative UI applications using Vercel AI SDK
- Next.js App Router with AI streaming
- AI-powered component libraries (ai-elements)
- Server-side rendering with AI generation
- Real-time AI chat interfaces
- Tool approval workflows
- Structured output generation

## Commands

- `/vercel-gen-ui` - Main team command for generative UI development
- `/status` - Check team status and active tasks

## Skills

- **vercel-deployment**: Vercel deployment best practices and configuration
- **nextjs-gen-ui**: Next.js patterns for generative UI
- **vercel-ai-sdk**: Vercel AI SDK implementation patterns
- **ai-elements**: AI-powered component development
